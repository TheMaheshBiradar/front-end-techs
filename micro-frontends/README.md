This repo contains microfront end example.

 1. **cooperative-bank-accounts** - This is react js based micro front end which contains GUIs.
 
 2. **cooperative-bank-payments** - This is react js based micro front end which contains GUIs.
 
 3. **cooperative-bank-container** - Container micro front end has it's own GUI and it embeds the other micofront end. See references to .env file
